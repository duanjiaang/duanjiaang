# vue带路由的模板项目

- 确保已经成功安装过nodejs和vuecli
- 第一次需要执行`npm install`初始化项目
- 每次开发项目需要执行`npm run serve`
- 执行`npm run build`可以打包项目，方便发布到正式服务器
- 使用模板项目的方法
  - 删除`node_modules`目录
  - 删除`dist`目录
  - 复制本文件夹到新位置
  - 修改复制后的文件夹名称为项目名称
  - 修改`package.json`中的项目名称和版本号
  - 修改`package-lock.json`中的项目名称和版本号
  - 执行上面说明的项目初始化过程

- 第三方js导入(npm方式)
  - `npm install --save qs`
  - `npm install --save axios`
